## Run (from `Main`)

Compile:
```bash
javac *.java
```

Run:
```bash
java Main
```

If using FlatLaf jar:
```bash
javac -cp ".:flatlaf-3.7.jar" *.java
java -cp ".:flatlaf-3.7.jar" Main
```
